Hover

Descripción

El :hover es un pseudo-clase en CSS que se activa cuando el usuario pasa el cursor sobre un elemento. Se usa comúnmente en enlaces, botones e imágenes para mejorar la interactividad.

Sintaxis

selector:hover { propiedad: valor; }

Color

Descripción

La propiedad color define el color del texto de un elemento. Se puede especificar usando nombres de colores, valores hexadecimales, RGB o HSL.

Sintaxis

selector { color: valor; }

Margin

Descripción

La propiedad margin establece el espacio exterior alrededor de un elemento. Se puede definir para cada lado (top, right, bottom, left) o en forma abreviada.

Sintaxis

selector { margin: valor; }

Ejemplo

Este cuadro tiene un margen de 20px alrededor de su contenido.

Text-Decoration

Descripción+

La propiedad text-decoration se usa para decorar el texto de un elemento. Se puede usar para subrayar, tachar, resaltar o cambiar el color del texto.

Sintaxis

selector { text-decoration: valor; }

Text-Align

Descripción

La propiedad text-align se usa para alinear el texto de un elemento. Se puede alinear a la izquierda, derecha, centrar o justificar.

Sintaxis

selector { text-align: valor; }
